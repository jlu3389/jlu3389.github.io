数学基础知识

一些数学基础知识

角度，度和弧度

“角度”度量平面中的旋转，常用希腊字母θ代表角度。最重要的两种角度单位是度（°）和弧度（rad） 弧度是基于园的一种单位。当用弧度表示两条射线的夹角时，我们指定了单位园上的一段圆弧。这个就是弧度 。

圆的周长是2π，π大于等于3.1415926。因此，2π代表旋转一圈。

因为360°=2π，180°=π，所以将弧度转成角度时乘以180/π（52.29578），角度转成弧度乘以π/180（0.01745329）。

三角函数

一个沿着X轴方向的向量，沿着逆时针旋转角度θ，那么这个旋转后的向量的x，y分别为。

x = cosθ

y = sinθ

另外还有一些三角函数

tanθ = sinθ/cosθ

secθ = 1 / cosθ

cscθ = 1 / sinθ

cotθ = 1 / tanθ = cosθ/sinθ

假设一个直角三角形，设短边为a，长边为b，斜边为c，角度θ为a与c相交的角度。则：

cosθ = a / c

secθ = c / a

sinθ = b / c

cscθ = c / b

tanθ = b / a

cotθ = a / b

注意 因为相似三角形的性质，即使斜边不是单位长度时，上面公式依然成立。但当θ为钝角时就不行了，因为不能构造一个内角为钝角的直角三角形。此时可以进行一些推广，用r代表向量长途，可得：

cosθ = x / r

secθ = r / x

sinθ = y / r

cscθ = r / y

tanθ = y / x

cotθ = x / y
